# Tailwind CSS Landing Page

## Project Overview

This project is a **simple landing page** built using **HTML and Tailwind CSS**.
The purpose of this project is to practice building modern UI layouts using Tailwind utility classes.

The landing page represents a coding education platform called **Shine Coding Schools** and includes a navigation bar, hero section, feature highlights, and a call-to-action button.

---

## Features

* Responsive layout using Flexbox
* Clean navigation bar
* Hero section with heading and highlighted text
* Feature list with check icons
* Call-to-action button
* Image section with full cover styling
* Built using Tailwind utility classes

---

## Technologies Used

* HTML5
* Tailwind CSS (CDN)
